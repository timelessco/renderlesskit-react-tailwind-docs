# Textarea

The Textarea component allows you to easily create multi-line text inputs.

<ComponentLinks github="textarea" theme="textarea" story="forms-textarea" />

## Imports

```js
import { Textarea } from "@renderlesskit/react-tailwind";
```

- **Textarea** - main textarea component

## Usage

Basic usage

<InteractiveCodeblock booleanProps={["isDisabled", "isReadOnly", "isInvalid"]}>
  {({ spreadProps }) =>
    `<Textarea placeholder="Give us your feedback"${spreadProps} />`
  }
</InteractiveCodeblock>

## Auto Resize

By setting autoSize prop the textarea will increase it's size according to the
content.

```jsx live
<Textarea autoSize />
```

## Resize direction

`resize` prop contarints the resize direction of the textarea, It can accept 3
values `both | vertical | horizontal | none`

```jsx live
<Textarea resize="none" defaultValue="Not Resizable" />
<Textarea resize="both" defaultValue="Resizable on both axis" />
<Textarea resize="vertical" defaultValue="Resizable only on Y axis" />
<Textarea resize="horizontal" defaultValue="Resizable only on X axis" />
```

## Cols and Rows

The following example show a textarea, with a set numbers of rows and columns
and some default content. Notice that even when autoSize is enabled the textarea
won't expand more than 5 rows.

```jsx live
<Textarea
  autoSize
  rowsMax="5"
  rowsMin="2"
  cols="50"
  defaultValue="Set numbers of rows and cols"
/>
```
