# Badge

Display an indicator that requires attention.

<ComponentLinks github="badge" theme="badge" story="primitives-badge" />

## Imports

```js
import { Badge } from "@renderlesskit/react-tailwind";
```

## Usage

<InteractiveCodeblock
  booleanProps={["floating"]}
  themeProps={{ size: "badge.size", variant: "badge.variant" }}
>
  {({ spreadProps, props }) => {
    const floating = outdent`
      <Button variant="outline">
        Hello world
        <Badge${spreadProps}>25</Badge>
      </Button>
    `;
    return props.floating ? floating : `<Badge${spreadProps}>Primary</Badge>`;
  }}
</InteractiveCodeblock>

## Badge Variants

Badge has 4 variants `primary` `secondary` `outline` `ghost`

```jsx live
<div className="space-x-2">
  <Badge variant="primary">primary</Badge>
  <Badge variant="secondary">secondary</Badge>
  <Badge variant="outline">outline</Badge>
  <Badge variant="ghost">ghost</Badge>
</div>
```

<Callout>
  You can add extra variants & sizes via the theme file. Checkout [theming
  guide](/docs/theming).
</Callout>

## Badge Sizes

Badge comes with 3 sizes `sm` `md` `lg`

```jsx live
<div className="space-x-2">
  <Badge size="sm">Small</Badge>
  <Badge size="md">Medium</Badge>
  <Badge size="lg">Large</Badge>
</div>
```

## Floating Badge

In some scenarioes where you want to add a floating badge on an element's
corners you can set the `Badge` component to be `floating`

```jsx live
<Button variant="outline">
  Hello world
  <Badge size="sm" position="top-right" floating>
    20
  </Badge>
</Button>
```
